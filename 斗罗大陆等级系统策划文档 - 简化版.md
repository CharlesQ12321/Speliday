## 1. 系统概述

### 1.1 设计目标

- **核心体验：** 通过单词拼写练习获得魂力值，提升斗罗大陆等级
- **简化机制：** 仅保留等级系统、日常任务、副本挑战三大核心玩法
- **单人体验：** 专注个人成长，无需社交

### 1.2 核心概念

| 概念 | 说明 |
|------|------|
| **魂力值** | 通过拼写练习获得的积分，用于升级 |
| **等级** | 从魂士到封号斗罗共 10 个大阶，100 个小级 |
| **修炼** | 单词拼写练习 |
| **魂骨** | 副本掉落的装备，提供僵尸对战增益 |

## 3. 魂力值获取规则

### 3.1 基础规则

**每次练习获得的积分 = 获得的魂力值**

- 练习结束时的总积分直接转换为魂力值
- 魂力值自动累加，达到升级条件即可升级
- **失败惩罚：** 练习失败（僵尸到达豌豆）不获得任何魂力值

### 5.2 任务规则

#### 5.2.1 突破任务阶段

- **解锁条件：** 玩家达到对应等级后解锁
- **首次挑战：** 标记为"突破任务"，显示红色标识
- **挑战限制：** 无限制，可反复挑战直到成功
- **突破奖励：** 获得对应魂环称号
- **成功后：** 突破任务转变为普通副本任务

#### 5.2.2 副本任务阶段

- **挑战条件：** 突破成功后即可挑战
- **挑战次数：** 每个副本每日可挑战 3 次
- **通关奖励：** 魂力值 = 练习积分 × 副本系数
- **额外掉落：** 魂骨（概率掉落，见上表）

### 5.3 副本任务要求

每个副本任务有特定的完成要求，难度递增。句子填空练习的单词数量最多为 50 个，正确率要求逐步提高：

- **任务要求：** 完成一次 50 个单词的拼写练习
- **正确率要求：** ≥65%
- **说明：** 基础单词拼写，适合新手入门

#### 5.3.2 史莱克学院
- **任务要求：** 完成一次 25 个单词的句子填空练习

#### 5.3.5 昊天宗

#### 5.3.6 海神岛

## 6. 魂骨系统

### 6.1 魂骨总览

魂骨是副本挑战掉落的特殊装备，装配后可获得僵尸对战增益技能。

**5 种魂兽 × 5 个部位 = 25 种魂骨**

### 6.2 魂骨部位

| 部位 | 数量 |
|------|------|
| 头骨 | 1 |
| 躯干骨 | 1 |
| 左臂骨 | 1 |
| 右腿骨 | 1 |
| 外附魂骨 | 1 |

**说明：** 魂骨部位不与属性绑定，每个魂骨获得时随机获得 5 种属性之一。

### 6.3 魂骨获得与鉴定

#### 6.3.1 获得魂骨

玩家获得魂骨时：
- **立即显示：** 魂骨部位 + 魂骨名称
- **未知信息：** 魂骨属性类型和属性值
- **状态标记：** 未鉴定

#### 6.3.2 魂骨鉴定

- **鉴定地点：** 魂骨图鉴界面
- **鉴定操作：** 点击未鉴定魂骨，选择"鉴定"
- **鉴定结果：** 显示魂骨的具体属性类型和数值
- **鉴定消耗：** 免费

### 6.4 5 种魂兽魂骨

每种魂兽的魂骨具有不同的套装技能，个体属性在鉴定时随机确定：

#### 6.4.1 曼陀罗蛇魂骨套装
- **套装技能：** 积分翻倍（练习获得的积分×2）

#### 6.4.2 柔骨兔魂骨套装
- **套装技能：** 失败重生（失败后不中断，可继续练习 1 次）

#### 6.4.3 邪眸白虎魂骨套装
- **套装技能：** 连击不断（连击中断后连击数减一，而非重置）

#### 6.4.4 幽冥灵猫魂骨套装
- **套装技能：** 答错不惩罚（答错时僵尸不前进）

#### 6.4.5 七宝琉璃魂骨套装
- **套装技能：** 失败获积分（失败获得 50% 积分）

### 6.5 魂骨属性

每个魂骨鉴定时随机获得 1 种属性，5 种属性的取值区间如下：

| 属性类型 | 属性名称 | 取值区间 |
|---------|---------|---------|
| **生命值** | 魂骨生命 | +600 ~ +1500 |
| **击退效果** | 魂骨力量 | +10% ~ +30% |
| **闪避率** | 魂骨闪避 | +6% ~ +18% |
| **防御值** | 魂骨防御 | +40 ~ +120 |
| **僵尸减速** | 魂骨减速 | -6% ~ -20% |

**说明：**
- 每个魂骨只能拥有 1 种属性
- 属性数值在对应区间内随机生成
- 同部位的不同魂骨可替换装备

### 6.6 套装效果

集齐同一魂兽的 5 个部位可激活套装技能：

| 套装名称 | 套装技能 | 技能说明 |
|---------|---------|---------|
| 曼陀罗蛇 | 积分翻倍 | 练习获得的积分×2 |
| 柔骨兔 | 失败重生 | 失败后不中断，可继续练习 1 次 |
| 邪眸白虎 | 连击不断 | 连击中断后连击数减一，而非重置 |
| 幽冥灵猫 | 答错不惩罚 | 答错时僵尸不前进 |
| 七宝琉璃 | 失败获积分 | 失败获得 50% 积分 |

### 6.7 魂骨装配规则

- **装配数量：** 最多同时装配 5 块魂骨（每个部位 1 块）
- **魂骨替换：** 可随时替换已装配的魂骨
- **套装识别：** 自动识别并激活套装技能

## 7. 玩家属性

### 7.1 玩家基础属性

玩家在拼写练习中具有以下属性：

| 属性 | 基础值 | 说明 |
|------|-------|------|
| **生命值** | 1000 | 被僵尸攻击时受到伤害，归零则失败 |
| **防御力** | 0 | 减少僵尸攻击造成的伤害 |
| **闪避率** | 0% | 概率闪避僵尸攻击，不受伤害 |
| **击退效果** | 100% | 答对单词时击退僵尸的距离 |

### 7.2 属性计算

```
最终属性 = 基础值 + 魂骨加成 + 等级加成

示例：
- 基础生命值：1000
- 魂骨加成：曼陀罗蛇头骨（+1000）+ 七宝琉璃头骨（+1150）
- 等级加成：魂宗 Lv.45（+500）

最终生命值 = 1000 + 1000 + 1150 + 500 = 3650
```

### 7.3 僵尸对战机制

#### 7.3.1 僵尸基础属性

| 属性 | 数值 | 说明 |
|------|------|------|
| **攻击力** | 100 | 每次攻击造成的基础伤害 |
| **移动速度** | 正常 | 每秒前进距离 |
| **血量** | 无限 | 僵尸无法被消灭 |

#### 7.3.2 战斗流程

- **答对单词：**
  1. 发射子弹击退僵尸
  2. 击退距离 = 基础值 × 击退效果
  3. 连击数 +1

- **答错单词：**
  1. 僵尸前进一段距离
  2. **僵尸发动攻击**，玩家受到伤害
  3. 连击数重置（或根据套装技能处理）

- **闪避触发：**
  1. 闪避率判定成功时，不受伤害
  2. 僵尸攻击被闪避，僵尸继续前进

#### 7.3.3 伤害计算

```
实际伤害 = 僵尸攻击力 - 玩家防御力

示例：
- 僵尸攻击力：100
- 玩家防御力：50
- 实际伤害 = 100 - 50 = 50 点

若玩家防御力提升至 120：
- 实际伤害 = 100 - 120 = 0 点（最低为 0）
```

#### 7.3.4 失败条件

- 玩家生命值归零时，练习失败
- 僵尸到达豌豆位置时，练习失败
- 失败后不获得任何魂力值（七宝琉璃套装除外）

- 表格中仅显示魂骨名称
- 不显示鉴定状态和属性值
- 未获得的魂骨以浅灰色显示

### 8.2 魂骨详情查看

玩家点击表格中某个魂骨格子后，弹出该类魂骨的详细信息列表：

```
┌─────────────────────────────────────────┐
│  蛇瞳冥骨（曼陀罗蛇·头骨）              │
│  已拥有：3 个                            │
├─────────────────────────────────────────┤
│  1. [已装备] [已鉴定]                   │
│     魂骨生命 +1000                      │
│     [卸下]                              │
├─────────────────────────────────────────┤
│  2. [未装备] [已鉴定]                   │
│     魂骨防御 +95                        │
│     [装备]                              │
├─────────────────────────────────────────┤
│  3. [未装备] [未鉴定]                   │
│     ？？？                              │
│     [鉴定]                              │
└─────────────────────────────────────────┘
```

**说明：**
- 显示该类魂骨的所有实例（不同属性值）
- 每个实例显示：
  - 装备状态（已装备/未装备）
  - 鉴定状态（已鉴定/未鉴定）
  - 已鉴定：显示具体属性类型和数值
  - 未鉴定：显示"？？？"
- 可进行的操作：
  - 已鉴定：装备/卸下
  - 未鉴定：鉴定
