# 魂骨作弊码使用说明

## 🎮 功能概述

在魂骨仓库搜索栏中输入特定作弊码，可以快速获得指定的魂骨。

## 📝 作弊码格式

### 1. 随机魂骨
```
hdhg
```
输入此代码将获得一个随机魂骨（随机选择魂兽类型和部位）

### 2. 指定魂骨
```
hdhgx-y
```
- **x**：列号（1-5）代表魂兽类型
- **y**：行号（1-5）代表部位

## 🎯 使用示例

### 示例1：获得邪眸白虎·躯干骨
1. 打开魂骨仓库
2. 在搜索栏输入：`hdhg3-2`
3. 点击搜索按钮
4. 系统将生成一个"邪眸白虎·躯干骨"

### 示例2：获得七宝琉璃·外附魂骨
1. 打开魂骨仓库
2. 在搜索栏输入：`hdhg5-5`
3. 点击搜索按钮
4. 系统将生成一个"七宝琉璃·外附魂骨"

### 示例3：获得随机魂骨
1. 打开魂骨仓库
2. 在搜索栏输入：`hdhg`
3. 点击搜索按钮
4. 系统将生成一个随机魂骨

## ⚠️ 注意事项

1. **大小写不敏感**：输入 `HDHG1-1` 或 `hdhg1-1` 效果相同
2. **必须先选择角色**：使用作弊码前需要先选择一个游戏角色
3. **范围限制**：x和y必须在1-5之间，超出范围将提示错误
4. **格式要求**：必须严格按照 `hdhgx-y` 格式输入
5. **自动鉴定**：生成的魂骨为未鉴定状态，需要鉴定后才能装备

## 🔧 技术实现

作弊码功能在 `app.js` 的 `searchSoulBones()` 函数中实现：

1. 使用正则表达式 `/^hdhg(\d)-(\d)$/` 解析作弊码
2. 验证列号和行号范围（1-5）
3. 映射到对应的魂兽类型和部位
4. 调用 `generateSpecificSoulBone()` 生成魂骨
5. 显示获得提示并刷新仓库页面
